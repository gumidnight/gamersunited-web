# Gamers United - Official Web Platform

## 🎮 Project Overview
The Gamers United web platform is a high-performance, esports-centric hub designed to unite the Cypriot gaming community. Built with a "Mobile-First" philosophy and a cutting-edge **Dark Esports Aesthetic**, the platform serves as the definitive source for news, tournaments, and official merchandise.

---

## 💎 Core Design System
- **Aesthetic**: Premium Dark Mode with Glassmorphism.
- **Palette**: Neon Purple (#7B2FF7), Neon Cyan (#00C6FF), and Deep Carbon surfaces.
- **Typography**: Bold, high-energy "Outfit" font with aggressive italic stylings for headlines.
- **Micro-Animations**: Framer Motion powered floating icons, pulsing "Live" indicators, and smooth page transitions.

---

## 🚀 Key Features

### 1. High-Impact Hero Hub
- **Dynamic Headline**: "YOUR GAME. OUR DOMAINE."
- **Visuals**: Animated neon scanners and floating gaming objects (controllers, trophies, shields).
- **CTA**: Integrated "Join the Gamers" (Discord) and "Latest News" portals.

### 2. Live Community Integration
- **Instagram "Social Spotlight"**: A custom-built, high-fidelity replica of premium Instagram feed plugins. Features real follower metrics (948+), the iconic IG gradient ring, and a tight 3x3 post grid.
- **Discord Status Sync**: The Navbar and Community sections feature a **Live Sync** indicator showing current online member counts via the Discord API.
- **Discord Widget**: Integrated official server widget nested within a custom-modeled iPhone container for mobile authenticity.

### 3. News & Media Portal
- **Breaking News Engine**: Supports "BREAKING" badges with a radar-pulse effect for high-priority stories.
- **Responsive Grid**: Interactive cards with hover-scaling cover images and category tags.
- **SEO Ready**: Dynamic slugs and metadata generation for every article.

### 4. E-Shop (Merch Central)
- **Printify/Printful Ready**: Backend integration prepared for synchronized merchandise lines.
- **UI**: High-detail product cards with glass backgrounds and price highlight accents.

### 5. Support & Legal Infrastructure
- **Interactive FAQ**: Categorized drop-down system for Tournaments, Membership, and Merch.
- **Legal Compliance**: Tailored Privacy Policy and Terms of Service focusing on community safety, fair play (anti-cheat), and data protection.

---

## 📱 Mobile Experience
The site is fully optimized for mobile devices (iPhone/Android), featuring:
- A custom "Glass-Strong" mobile navigation menu.
- Touch-optimized interaction zones.
- Performance tuning (92+ Lighthouse score) for fast loading on 4G/5G connections.

---

## 🛠️ Technology Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
